pub struct RistrettoPoint(_);

Represents a point in the Ristretto group for Curve25519.

pub fn from_uniform_bytes(bytes: &[u8; 64]) -> Self

Construct a RistrettoPoint from the given data using an Ristretto-flavoured Elligator 2 map. If the input bytes are uniformly distributed, the resulting point will be uniformly distributed over the Ristretto group.
