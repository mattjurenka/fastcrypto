pub fn prove_bit_length( value: u64, blinding: [u8; 32], bits: usize, domain: &'static [u8] ) -> Result<(PedersenCommitment, Self), FastCryptoError>

Prove that the value is an unsigned integer with bit length bits, this is equivalent to proving that the value is an integer within the range [0, 2^bits) Function only works for bits = 8, 16, 32, 64.

source

pub fn verify_bit_length( &self, commitment: &PedersenCommitment, bits: usize, domain: &'static [u8] ) -> Result<(), FastCryptoError>

Verifies that commitment is a Pedersen commitment of some value with an unsigned bit length bits. This is equivalent to proving that the value is an integer within the range [0, 2^bits) Function only works for bits = 8, 16, 32, 64.
